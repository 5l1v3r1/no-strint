python source code obfuscator. only obscures strings and integers

>> installation
   $ git clone https://github.com/zevtyardt/no-strint
   $ cd no-strint
   $ python2 setup.py install or pip install .
   $ strint -h
   
   # via pip, 1.4.5
   $ pip2 install strint
